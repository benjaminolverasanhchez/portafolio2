Propiedad Outline

Es una línea que se dibuja alrededor de los elementos, FUERA de los bordes, para que el elemento "resalte".

Propiedad Outline Color

Color del Contorno **Se utilizo la propiedad outline-style:solid**

Propiedad Outline width

Medida del Contorno**Se utilizo la propiedad outline-style:solid**

Propiedades: OUTLINE-STYLE

dotted Outline // Contorno Punteado Cuadrados
dashed Outline // Contorno Punteado Rectangulos
solid Outline // Contorno Solido
double Outline // Doble Contorno
groove Outline // Ranura dentro del Contorno
ridge Outline // Ranura fuera del Contorno
inset Outline // Degradado dentro del Contorno
outset Outline // Degradado fuera del Contorno
